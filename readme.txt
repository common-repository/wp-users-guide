投稿編集画面に使用方法のヘルプを表示させるプラグインです。

== Description ==

I'm sorry. This plugin is japanese only...

このプラグインは主に「投稿者」や「寄稿者」などの初心者ユーザーをターゲットにした、記事編集のヘルプを表示させるプラグインです。
編集画面の各項目にツールチップを表示させたり、（自動で追加される）ヘルプアイコンをクリックして詳細なヘルプを表示させる事ができます。

また、ダッシュボード画面に『ようこそ画面』を表示させ、迷う事なく投稿記事が書けるようにもしています。

ver 1.01 投稿保存時、カテゴリーが選択されていない場合、確認画面を表示する機能を追加しました。

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-upsers-guide` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.


== Frequently Asked Questions ==

= 消してしまったダッシュボードの『ようこそ画面』を復活させる方法。 How to revive the "Welcome Screen" in the dashboard =

管理画面の『設定』→『ユーザーマニュアル』で設定を変更させる事ができます。
You can change setting the plugin through the 'Setting' → 'User Manual' screen in WordPress.
